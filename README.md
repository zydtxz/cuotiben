# 错题本

给孩子用的错题整理小工具，完全单文件 Web 应用，双击 `index.html` 即可打开。

## 主要功能

- 错题卡片：题目、解析、知识点、难度
- 手写板：分析区 / 正确区 / 知识点区，支持缩放、平移、撤销、清空
- 图形与符号：直线、矩形、椭圆、箭头、常用数学符号
- 图片插入：可拖拽、缩放、旋转的错题截图
- OCR 识别：拍照/截图后识别文字填入题目
- PDF 导出：A4 排版，一键生成并下载

## 在平板/手机上使用

1. 用浏览器打开 `index.html`（推荐 Edge / Safari / Chrome）。
2. 可以“添加到主屏幕”，像 App 一样使用。
3. 数据保存在浏览器本地，**换设备或清缓存会丢失**，重要内容请定期导出 PDF 备份。
